# vrmap Demo

花火大会の座席からの景色を、事前に360°パノラマで確認できるWebサービスです。
https://vrmap-rho.vercel.app/
---

## 背景

花火大会のチケット購入時、**座席からの見え方がわからない**という問題がありました。

既存の座席紹介サイトでは以下の課題が挙がっていました。(https://suwako-hanabi.com/kojyou/tickets/citizens-first/)

- 木や建物などの障害物で花火が見切れてしまう
- 購入後に「思っていた眺めと違う」というクレームが発生する
- **実際の現地との乖離**があり、サイトの情報と現地の状況が一致しないケースがある
- **静止画による場所の写真のみ**の掲載のため、実際の見え方や雰囲気が十分に伝わらない

こうした課題を解決し、サービスの質を向上させるために本プロジェクトを立ち上げました。

---

## サービス概要

**Insta360** で各座席エリアを撮影した360°パノラマ画像を、会場マップ上のピンに紐付けて公開します。
来場者はチケット購入前にピンをタップするだけで、実際の座席からの眺めを体感できます。

---

## 機能

- 会場マップ上にエリアごとのピンを表示
- ピンをタップすると360°パノラマビューアが起動（Pannellum使用）
- ドラッグ／スワイプで視点を自由に動かせる
- スマホ（iPhone）対応のレスポンシブレイアウト

```
    └── ...
```

## エリアの追加・編集

`index.html` 内の `AREAS` 配列を編集します。

```js
let AREAS = [
  {
    "id": "food",
    "label": "フードコートエリア",
    "x": 12.6,           // マップ上のX位置（%）
    "y": 37.0,           // マップ上のY位置（%）
    "desc": "エリアの説明文",
    "panoUrl": "images/food_area.jpg",  // 360°画像のパス
    "pitch": 0,          // 初期上下角度
    "yaw": 0,            // 初期左右方向
    "hfov": 100          // 初期視野角
  },
  ...
];
```

### 360°画像について

Pannellumは **等正距円筒図法（Equirectangular）** 形式の画像に対応しています。
Insta360で撮影した画像はそのまま使用できます。

## デプロイ

[Vercel](https://vercel.com/) にリポジトリを連携してプッシュするだけで自動デプロイされます。

```bash
# ローカル確認（Live Server推奨）
# VS Code拡張「Live Server」でindex.htmlを開く
# → http://localhost:5500
```

> ⚠️ `file://` で直接開くと360°画像が読み込まれない場合があります。必ずLive ServerまたはVercel経由で確認してください。
